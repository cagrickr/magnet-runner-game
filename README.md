# 🎮 Magnet Runner

A 2D platform game built with **Pygame** and **SQLite**, featuring magnetic-based gameplay mechanics.

---

## 🧲 Gameplay

Magnet Runner is a physics-inspired platform game where the player controls a magnet character and navigates through levels filled with obstacles.

### Core Mechanic:
- Opposite poles → gain points  
- Same poles → lose points  
- Requires timing and strategy  

---

## 🧩 Features

- 🏆 Leaderboard system  
- 👤 Player profiles  
- 📈 Level progression  
- 📍 Checkpoint system  

---

## 🛠️ Tech Stack

- Python  
- Pygame  
- SQLite
